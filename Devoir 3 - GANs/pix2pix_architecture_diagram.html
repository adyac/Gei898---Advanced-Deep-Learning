U-Net Generator: 4 encoder blocks compress spatial information while expanding feature dimensions (64→128→256→512). The bottleneck captures global context at 8×8 resolution. 4 decoder blocks expand spatial dimensions back while concatenating skip connections from each encoder level. This preserves fine spatial details essential for accurate mask localization.

Encoder Blocks

Progressive 2× downsampling with Conv-BN-ReLU and MaxPool operations. Doubles channels at each level.

Skip Connections

Concatenate encoder features to corresponding decoder blocks. Preserves fine-grained spatial information from each level.

Bottleneck

512 channels at 8×8 resolution. Represents the most compressed, context-rich feature representation.

Output Activation

Sigmoid function: outputs range [-1, 1] matching the ground truth normalization for loss computation.

PatchGAN Discriminator: Instead of classifying the entire image, PatchGAN outputs a 7×7 grid of patch-level scores. Each cell independently predicts whether a 16×16 patch is real or fake. This encourages fine-grained realism at the local level rather than global coherence, which is ideal for detailed mask generation.

Input Concatenation

Condition (3 channels) + Image (3 channels) = 6 channels. Allows discriminator to evaluate correspondence.

Progressive Downsampling

4 Conv blocks with 2× downsampling reduce resolution from 128×128 to 8×8, creating receptive field of ~70×70.

Patch-Based Output

7×7 grid of scores (not a single binary classification). Captures local realism across the entire image.

Dropout Regularization

0.3 dropout on layers 3-4 prevents discriminator overfitting and ensures stable adversarial training.

Training Loop: Each epoch: 1) Load a batch of paired images and masks 2) Generate fake masks using current generator 3) Evaluate both real and fake masks with discriminator 4) Compute combined loss (HingeLoss + 2.0 × reconstruction) 5) Update discriminator first, then generator 6) Evaluate on validation set after each epoch 7) Repeat for 500 epochs with early stopping if validation loss plateaus

Loss Components:
  • Hinge Loss: max(0, 1 - D_fake) for adversarial training. More stable than BCE for GANs.
  • BCE Loss (2.0×): Binary cross-entropy for pixel-level classification (defect vs non-defect).
  • L1 Loss (0.5×): Encourages spatial smoothness while reducing bias toward gray outputs.
  • Label Smoothing (α=0.2): Real labels = 0.8 instead of 1.0, prevents discriminator overfitting.
